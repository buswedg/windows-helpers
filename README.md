# Windows Helpers

- App-Downloader-Installer: Downloads and/or installs applications using PowerShell.
- NVIDIA-Driver-Manager: Downloads and/or installs essential Nvidia display drivers using PowerShell and 7-Zip.
- Store-App-Uninstaller: Uninstalls Windows Store applications using PowerShell.
- WinGet Installer: Installs applications using WinGet and PowerShell.
