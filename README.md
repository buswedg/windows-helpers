# Windows Helpers

- **App-Installer**: Downloads and/or installs a configurable list of applications using PowerShell.
- **Background-Scheduler**: Automatically change desktop wallpapers based on the time of day.
- **Defender-Excluder**: Applies a configurable list of directory exclusions to Windows Defender using PowerShell.
- **Network-Adapter-Manager**: Configure a set of network adapters using Powershell.
- **Store-App-Uninstaller**: Uninstalls a configurable list of Windows Store applications using PowerShell.
- **Wake-On-Lan**: Starts physical machines using Wake On LAN magic packets.
- **WinGet-Installer**: Installs a configurable list of applications using WinGet and PowerShell.
